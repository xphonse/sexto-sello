## ComparaciГіn de NavegaciГіn

| Aspecto | Website | Mobile App |
|---------|---------|------------|
| **Tipo** | File-based routing | Stack Navigator |
| **Back button** | Browser back | Native back + custom |
| **URLs** | Visibles y compartibles | No hay URLs |
| **Deep linking** | Directo | Requiere configuraciГіn |
| **State** | Stateless | Redux persiste |
| **Transitions** | Client-side routing | Native animations |
